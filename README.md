# Spinix — Issue Tracker

This repository is the public issue tracker for **Spinix**, a fast and modern spin-the-wheel app.

👉 **App:** [spinix.nowakdev.com](https://spinix.nowakdev.com)

---

## What is Spinix?

Spinix is a browser-based spin-the-wheel tool — no account required, no tracking, no ads.  
Open the page, add your entries, and spin.

**Features:**
- Custom wheel entries with auto-save in the URL (shareable links)
- Templates for common use cases (classrooms, giveaways, team picks, …)
- Live sessions — share a read-only view with real-time spin animations
- Light / dark mode
- English & German UI

---

## Reporting an issue

Found a bug or have a feature request? [Open an issue](../../issues/new) and include:

- **Bug:** what you did, what you expected, what happened instead — browser and OS if relevant
- **Feature request:** what you'd like and why it would be useful

Please search [existing issues](../../issues) before opening a new one.
